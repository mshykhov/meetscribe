# Meetscribe

Automated meeting video processing on macOS Apple Silicon. Drop a video, get a transcript with speakers and an AI summary.

## How it works

Video in watch folder -> MLX-accelerated transcription -> word alignment -> speaker diarization -> Claude summary -> organized output folder

```
{date}-{topic}/
├── video.mp4
├── transcript.txt     # [00:15] SPEAKER_00: text...
└── summary.md         # topic, decisions, action items
```

Date comes from video metadata. Topic is generated by Claude.

## Requirements

- macOS with Apple Silicon (M1+)
- Python 3.12+
- [HuggingFace token](https://huggingface.co/settings/tokens) with accepted licenses for [speaker-diarization](https://huggingface.co/pyannote/speaker-diarization-community-1) and [segmentation](https://huggingface.co/pyannote/segmentation-3.0)
- [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code) for summary generation

## Quick start

```bash
brew install python@3.12 ffmpeg terminal-notifier
brew install --cask swiftbar  # optional, menu bar status

git clone https://github.com/mshykhov/meetscribe.git && cd meetscribe
python3.12 -m venv .venv && .venv/bin/pip install -e .
.venv/bin/pip install "senko @ git+https://github.com/narcotic-sh/senko.git"

cp .env.example .env  # set HF_TOKEN, adjust paths
./scripts/install.sh install
```

## Usage

**Automatic:** drop video into watch folder. **Manual:** `.venv/bin/python -m src.process /path/to/video.mp4`

```bash
./scripts/install.sh health      # check status
./scripts/install.sh retry       # retry failed
./scripts/install.sh reprocess /path/to/video.mp4
```

## Configuration

Key `.env` options: `HF_TOKEN` (required), `WATCH_DIR`, `OUTPUT_DIR`, `WHISPER_MODEL` (tiny/small/medium/large-v2), `LANGUAGE`, `CLAUDE_MODEL`.

Recommended model: `medium` (best quality/speed balance on Apple Silicon).

## Stack

- **WhisperX-MLX** - transcription with MLX GPU acceleration
- **MLX wav2vec2** - word-level timestamp alignment
- **Senko** - speaker diarization via CoreML/Neural Engine
- **Claude CLI** - AI summary generation
- **launchd** - background service watching for new videos
- **SwiftBar** - menu bar processing status
- **terminal-notifier** - desktop notifications
